Create Your LED Formula

Enter formulas using reverse Polish notation for each color channel (Red, Green, Blue).

Available Functions:

  • cos(x), sin(x), tan(x)
  • acos(x), asin(x), atan(x)
  • sqrt(x), exp(x), abs(x)
  • pow(x, y)

Variables:

  • x: Position around hat (-1 to 1, 0 is front)
  • y: Height (0 to 1, bottom to top)
  • t: Time in seconds since start

Example:

t x + sin equals sin(t + x)

This creates a moving sine wave around the hat.
